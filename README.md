# react-boilerplate
React Project Setup (stylus, Babel). Note that all dependencies that you include should be included manually when configuring your project.

# Install

With npm:
```
npm install
```
With yarn:
```
yarn install
```
# Development
With npm:
```
npm run dev
```
With yarn:
```
yarn dev
```

This will run at localhost:8888
